Introduction

This guide will help you hack your Nintendo Switch.

Requirements:

  • SD Card - (128GB+ Recommended)
  • Computer or Android device
  • Internet Connection
  • RCM jig
  • USB-C cable

Is Your Switch Patched?

  1. Go to Settings -> System -> Serial Information
  2. Check to see if your switch is patched using IsMySwitchPatched
  3. If your switch is patched, you will need to modchip your switch.

Instructions:

  1. Put your Switch into RCM mode using the RCM jig and pressing Power and Volume Up Button (+)
  2. Connect your Switch to your computer or Android device using the USB-C cable.
  3. Run the exploit using TegraRcmGUI if you are on Windows or Rekado if you are on Android
  4. Navigate to Tools -> Partition SD
  5. Set the emuMMC (RAW) partition to 29 FULL
    • If you want to install Android or Linux afterwards, you can set the sliders for Android (USER) or Linux (EXT4) to at least 16GB
  6. Navigate to Next Step and then press Start in the menu that appears.
    • For Android select Legacy for Android 10/11 and Dynamic for the latest versions. Legacy and Dynamic are NOT intercompatible.
  7. From the Home menu, navigate to emuMMC -> Create emuMMC -> SD Partition -> Part 1.
  8. Wait for it to complete.
  9. Close the menu by pressing Close then go to Change emuMMC -> SD RAW 1 and press the Close button twice to return to the home menu.
  10. Once done, go to Tools -> USB Tools -> SD Card and plug your Switch into your PC using the USB-C cable you used before.

SD Card Preparation

  1. Download the latest version of DeepSea.
  2. Extract the contents of the zip file to the root of your SD card.
  3. Eject the switch and press Close.

Backups - Don't Skip

  1. Go to Tools -> Backup eMMC.
  2. Tap on eMMC BOOT0 & BOOT1.
    • The process should only take a few seconds, but if your SD card is slow, it may take around a minute.
  3. Tap Close to continue, then tap on eMMC RAW GPP.
  4. Go to the top right and press Close -> Home.
  5. Navigate to Tools -> USB tools -> SD card and plug your Switch into your PC via USB.
  6. Copy the backup folder on your microSD card to a safe location on your PC.
  7. Once done, you may delete the backup folder from the root of your microSD card and eject the UMS device safely from within your computer's operating system, then return to the Home menu.

Booting into emuMMC

  1. From the Home menu, navigate to the Launch menu.
  2. Press on Atmosphere FSS0 emuMMC and launch it.
  3. Once booted, check Settings -> System and see if it says AMS next to the version number.

Launching Homebrew Meu

  1. You can access it by opening the albums or by holding the R button while opening a game or app. If R isnt held, the game or app will launch like normal.
